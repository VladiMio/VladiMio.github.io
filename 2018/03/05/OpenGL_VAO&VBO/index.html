    VBO(Vertex Buffer Object，顶点缓冲对象),可以认为是抽象的显存对象，其中包含着顶点(一个或多个)的位置和着色信息，类似：

    但是当数据存储在VBO中时，OpenGL并不知道每一部分所代表的具体含义，因此需要借助VAO(Vertex Array Object，顶点数组对象)的帮助，即VAO中存储的数据规定了OpenGL如何理解VBO中的数据。

    通过定义VBO里面数据存放的方式、VAO的格式以及合适的GLSL 语言，就能够灵活地画出我们所需要的各种图形。
